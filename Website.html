HTML

HTML (Hypertext Markup Language) is a special hypertext language for creating any document or page that can be read by a browser. HTML is a fundamental base technology of the Internet and is a language based on descriptors that define the type and function of a web page. Reports collected on all web destinations and shown on the screens of computers associated with the Web are composed in HTML program code. HTML - permits you to put content squares, pictures on pages, make tables, select colors for the archive and the content within the report, include interactive media components, make hyperlinks, and joints between all these components.

JS

Javascript is a dynamic programming language mainly used in web browsers. JavaScript is a prototype-based scripting language with dynamic types and first-class functions. It has object-oriented, imperative, and functional programming principles. The writing style of JavaScript is derived from the C programming language. Its semantics and design are inspired by the Self and Scheme programming languages.
